Introduction

Elements and Components shown in this demo have been designed to be modular for easy integration into your own Web Projects.

Core Tokens are easily customisable in the CSS and you only need to import the elements you require into your JS.

This demo doubles as my portfolio as a junior web designer and developer. Showcasing my abilities in design, functionality and flexibility
Check out the 'About vCore' page to find out more!

The vCoreUi Library is and always will be open source as all good tools should be.
Enjoy!

Cursor Particle Effect

The vCoreUI cursor effect adds a smooth, multi coloured, wispy particle trail to the mouse pointer.
Toggle the switch below to activate or deactivate the effect.
Note: The effect auto-pauses while scrolling for performance.

window.vcoreCursorFlow = new vCoreCursorFlow();

Cursor Effect - Off

Scroll Sequence Video

The scroll sequence video effect plays a videos frame sequence based on the scroll position of the page.
Frames for the video will play in sync with the scroll / page movements, creating an engaging visual effect.
Adjust size and positions as needed in CSS.

class="scroll-sequence"
data-zip="assets/video/frames.zip"
data-frames="282"

Frame Packager

This Library also come with my Frame_Packager.py Python script.
Place .MP4 Videos in the root folder and run Frame_Packager.py to convert the .MP4 into individual frames saved as .WEBP images all packaged into a zip folder ready to be used as an asset for the Scroll Sequence Videos.

Refer to the README.md documentation provided with the packager to see full requirement needs and uses.
