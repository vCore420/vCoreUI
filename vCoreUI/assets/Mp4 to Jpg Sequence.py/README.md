## Mp4 to Jpg Sequence Converter in Python

## step 1

- make sure python 3x is installed
https://www.python.org/downloads/

- install opencv-python

pip3 install opencv-python

## step 2 

- place mp4 in folder with python script 'convert.py'
- run convert.py
- all jpgs of each frame will be saved in a folder in the same directory all lebeled frame_000*
